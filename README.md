# TOML.jl validation

This is test suite for validation of [TOML.jl](https://github.com/wildart/TOML.jl) package borrowed and addoped from Rust [toml-rs](https://github.com/alexcrichton/toml-rs) package.

**Requirements**:

- Julia v1.0.0
- TOML v0.4.0
- JSON
- Dates (stdlib)

**Run**:

```
julia toml-validator.jl
```
